## 5. 回归点（给测试 / Phase D）

### 门禁级（P0 相关，必过）

1. **换链一致**：结果肖像能出时，保存长图肖像行为一致；云失败/无网 → 可出文字结果，不白屏、不永久 Loading。  
2. **云函数面**：禁用 type（openid/CRUD/小程序码）调用失败；合法 `quiz/` 素材仍可换链。  
3. **白名单**：传入非 `quiz/` File ID → 拒绝。  
4. **计分分布**：跑 A6 建议的 Node Monte Carlo（≥5k）满足覆盖/占比门槛；抽查索尼娅/梅什金/地下室人可达。  
5. **合规脏词**：角色字段全文检索无医疗隐喻 P0；结果页 + 分享图有娱乐免责。  
6. **收集闭环**：测完 →「去看看」→ 图鉴可见本场新卡；重复掉落无假「新」；暗影不入册文案正确。  
7. **关键路径**：冷启动 → 20 题 → 结果 → 分享好友 / 长图 → 我的图鉴 → 再测（图鉴累计不丢）。  
8. **边界**：上一题改选结果变化；飞行模式可答完；云函数未部署时流程可走完。

### 体验级（若做了对应 P1）

9. 连续答 5 题，flash 无明显「等动画才能继续」烦感。  
10. 开始页首屏：标题 + 钩子 + CTA，不被 meta 卡片压过。  
11. 返回上一题选项回显。  
12. 社区占位有回流 CTA，无「坏了」感。

## 6. 冲突裁决说明

| 冲突 | 各方主张 | **裁决** |
|------|----------|----------|
| **Flash 打断** | A2：缩短/减频为 P0 体验；A6：不改玩法骨架；A3：闪卡沉浸是前半回路 | **本迭代降为 P1-1**：减弱时机与时长，**不删除闪卡机制、不改为全解锁**。理由：安全/换链/计分/脏词优先占满 P0 带宽；flash 属节奏优化，非阻断发布。 |
| **算法同质化 vs 题库大改** | A6 根因含 REF + 选项能量 + 角色簇；全修面大 | **P0-5 可控改动**：优先 `REF`/按轴 min–max 归一化；仅在不达标时微调少量选项轴值。**不做**整库选项重写、不做亲和分、不大改角色模板（留给后续迭代）。 |
| **结果页信息量** | A2 要减负；A3 要掉落说明 + 去图鉴；A7 要加免责 | **加短文案与入口，不堆新模块**：掉落规则 1–2 句 +「去看看」+ 底部免责；雷达/分析折叠属 P1，可与减负同做。 |
| **云函数双份** | A1 收敛；A5 加固两处防漏 | **P0 同时加固** `quickstartFunctions` 内嵌换链与独立 `getAssetUrls`（或删一留一）。合并实现可排 P2，但**不能只加固被调用的那份而漏掉另一份部署面**。 |
| **分享换链** | A1 架构债 / A4 风险 R1 / A5 旁路白名单 | **合并为单一 P0-1**，与 P0-3 白名单联动验收。 |
| **文学黑暗 vs 合规** | A5/A7 要提示；策划要保调性 | **改脏词 + 壳层免责/题材提示**；不删情节、不加「请就医」。 |
| **社区 Tab** | 空壳伤 IA；删除成本高 | **不删 Tab**；P1 预告卡 + 回流 CTA。 |
| **「人格」品牌词** | A7 要降诊断腔 | **保留玩法与品牌词**；改壳标题与个别脏词，不抹掉「人格」产品定位。 |

## 7. 给 Phase C 三主审的一句话

本迭代最小闭环 = **云函数加固（收口+白名单）+ 分享换链对齐 + 隐私披露 + 归一化治同质化 + 三处医疗脏词 + 结果免责 + 掉落→图鉴出口**；体验向 flash/开始页/结果减负进 P1，防范围膨胀项一律不做。
